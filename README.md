# 🍽️ Responsive Recipe Page

A professionally designed and fully responsive recipe page built using HTML and CSS. This project showcases a clean layout for displaying recipe information, including ingredients, preparation time, cooking instructions, and nutritional details.

## 📖 Overview

The Responsive Recipe Page is a front-end web development project that demonstrates the use of semantic HTML and modern CSS techniques to create an attractive and user-friendly recipe interface. The design adapts seamlessly to different screen sizes, ensuring a consistent experience across mobile, tablet, and desktop devices.

## ✨ Features

- Responsive design for mobile, tablet, and desktop screens
- Clean and modern user interface
- Recipe image section
- Preparation and cooking time details
- Ingredients list
- Step-by-step cooking instructions
- Nutrition information table
- Semantic and accessible HTML structure

## 🛠️ Technologies Used

- HTML5
- CSS3
